ModBot

ModBot is a moderation bot that contain everything you need for your server.

Features

FEATURE 1

ModBot contain a very cool "join-to-create" system that allow you to have private vc without creating 100 channels

FEATURE 2

ModBot contain a very useful ticket system that can temporary close a ticket so the user that created it cannot see it

FEATURE 3

ModBot contain a nice verification system based on captcha so you server will be in security

FEATURE 4

ModBot contain many games to play with your community and make it more active
